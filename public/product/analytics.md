# Analytics & Reporting - Data-Driven Insights

## Overview

Pre-built dashboards and custom reports to track team performance, measure AI effectiveness, and quantify ROI from support automation.

## Key Metrics

### Team Performance
- **First Response Time**: Average time from message received to first agent reply
- **Resolution Time**: Average time from conversation start to closure
- **Volume Trends**: Conversation count over time by channel, tag, or team
- **Agent Activity**: Conversations handled per agent with quality scores
- **CSAT Scores**: Customer satisfaction ratings with trend analysis

### AI Effectiveness
- **Deflection Rate**: % of inquiries resolved by AI without human escalation (target: 60-70%)
- **Autonomous Resolution**: Conversations closed by chatbot without agent involvement
- **Escalation Reasons**: Why AI handed off to humans (low confidence, customer request, etc.)
- **Cost Savings**: Calculated based on deflected conversations Ã— avg handling cost
- **AI vs Human CSAT**: Compare customer satisfaction for AI vs agent-handled conversations

### Business Impact
- **ROI Calculator**: Track savings from AI automation vs hiring additional support staff
- **Channel Distribution**: See which channels customers prefer
- **Peak Hours**: Identify high-volume times for staffing optimization
- **Tag Analysis**: Most common issue types to inform product/content improvements
- **SLA Compliance**: % of conversations meeting response/resolution time targets

## Dashboards

### Overview Dashboard
Real-time snapshot of:
- Conversations handled today/this week/this month
- Current queue depth
- Average response & resolution times
- CSAT score
- Top performing agents

### AI Performance Dashboard
Detailed AI metrics:
- Deflection rate trends
- Autonomous vs escalated conversations
- Common escalation patterns
- Knowledge base gap analysis
- Credit usage tracking

### Team Dashboard
Agent-level performance:
- Conversations handled per agent
- Average response/resolution times
- CSAT scores by agent
- Auto-QA scores (Pro plan)
- Workload distribution

### Custom Reports
Build your own reports with:
- Date range selection
- Filter by channel, agent, tag, status
- Group by any dimension
- Export to CSV/Excel
- Schedule email delivery

## Data Export

- **CSV/Excel Export**: Download any report for custom analysis
- **API Access**: Pull raw data via REST API for BI tools
- **Scheduled Reports**: Auto-email reports daily/weekly/monthly
- **Data Retention**: 12 months on Standard, unlimited on Pro

## Integrations

Connect analytics to external tools:
- **Google Analytics**: Track chatbot deflection impact on website metrics
- **Tableau/Looker**: Import Pullse data into your BI platform
- **Slack**: Daily digest of key metrics posted to team channel
- **Email Notifications**: Alert when SLAs breached or CSAT drops below threshold

## Real-time Monitoring

- **Live View**: See current conversation queue and active agents
- **Alerts**: Configure notifications for anomalies (spike in volume, CSAT drop, SLA breach)
- **Performance Targets**: Set goals and track progress in real-time

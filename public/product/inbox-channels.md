# Inbox & Channels - Unified Communication Hub

## Overview

Pullse consolidates email, live chat, voice, and social messaging into one unified inbox with real-time collaboration features and intelligent routing.

## Supported Channels

### Email (Available Now)
- Connect unlimited email addresses
- Shared inbox with automatic threading
- Rich text composer with attachments
- Template/macro support
- Signature management per user

### Live Chat (Available Now)
- Embeddable widget with custom branding
- Proactive messaging triggers
- Real-time typing indicators
- File sharing and screenshots
- Offline message capture

### Voice (Coming Q1 2025)
- Inbound/outbound calling
- AI call transcription
- Call recording with playback
- IVR integration
- Click-to-call from tickets

### Social Messaging (Coming Q2 2025)
- WhatsApp Business API
- Facebook Messenger
- Instagram DM
- Twitter/X DM integration

## Unified Inbox Features

### Real-time Collaboration
- **Collision Detection**: See when teammates are viewing the same conversation
- **Internal Notes**: Leave context for other agents (customer doesn't see)
- **@Mentions**: Tag teammates for input or escalation
- **Conversation Assignment**: Claim tickets or assign to specific agents
- **Status Tracking**: Open, pending, solved with custom statuses

### Smart Organization
- **Unified View**: All channels in one feed (no app switching)
- **Filtering**: By channel, priority, status, assignee, or tag
- **Search**: Full-text search across all conversations and attachments
- **Folders/Views**: Custom inbox views per team or use case
- **Priority Flagging**: Mark urgent conversations for immediate attention

### Automation
- **Auto-routing**: Assign conversations based on skills, language, or workload
- **SLA Timers**: Track response and resolution time with escalation triggers
- **Canned Responses**: Insert pre-written replies with keyboard shortcuts
- **Smart Suggestions**: AI recommends relevant macros based on conversation context

## Customer Context

Every conversation shows:
- Customer profile (name, email, company, custom fields)
- Conversation history across all channels
- CRM data (from Salesforce, HubSpot, etc.)
- Purchase history (from Shopify, Stripe, etc.)
- Open/past tickets
- Custom properties and tags

## Mobile Experience

- Fully responsive web interface (works on any device)
- Push notifications for new messages
- Native iOS/Android apps coming 2026
- Offline mode with sync when connection restores

## Channel Analytics

- Volume by channel
- Response time per channel
- Customer preference trends
- Channel-specific CSAT scores
- Peak hours by channel
